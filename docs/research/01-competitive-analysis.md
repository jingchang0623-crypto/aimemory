## 二、核心竞品分析

### Chat Memo（最直接竞品）

- 创始人：Eze（中国AI博主）
- 定位："AI Memory Hub"
- 支持平台：ChatGPT、Gemini、DeepSeek、Claude、豆包、Kimi、通义千问
- 数据存储：100%本地，隐私优先
- 定价：核心功能永久免费，未来云同步和AI洞察将收费
- 弱点：无AI分析能力、无跨平台记忆注入、无移动端

### Echoes

- 定价：Free(20次搜索/月) → Plus $8.99/月 → Lifetime $199.99
- 功能亮点：跨LLM搜索、对话摘要、批量操作
- 弱点：免费版限制极严（20次搜索/月），用户评价中等（4.0分）

### AI Context Flow

- 定价：Free(100cr/天) → Plus $10/月 → Pro $20/月
- 核心差异化：Context Buckets（可复用上下文块）
- 弱点：仅2000用户，产品成熟度存疑

## 四、竞品空白与机会

1. **AI主动分析记忆** — 竞品仅保存，无分析
2. **真正的跨平台记忆注入** — 仅AI Context Flow尝试，用户仅2K
3. **中英文双语市场** — 中国AI平台（豆包/Kimi/通义千问）需求未满足
4. **记忆的"活性"管理** — 所有产品都是"保存+检索"模式

## 五、结论

市场碎片化但活跃，30+产品竞争但没有一个同时做好"保存+组织+AI分析+跨平台注入"。核心机会在于**让记忆从被动存储变为主动智能**。
