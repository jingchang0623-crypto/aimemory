🧠 AI Memory Manager — Your AI conversations, searchable forever.

Never lose an important AI conversation again. AI Memory automatically saves your chats from ChatGPT, Claude, DeepSeek, and Gemini, making them instantly searchable across all platforms.

## Features

✅ **Auto-Save Conversations**
Automatically captures conversations as you chat — no manual action needed. Every message saved in real-time.

✅ **Full-Text Search**
Search across all saved conversations instantly. Find that one solution from months ago in seconds.

✅ **Multi-Platform Support**
Works with ChatGPT, Claude, DeepSeek, and Gemini. One extension, all your AI conversations in one place.

✅ **100% Private Storage**
All data stored locally in your browser session. No accounts, no cloud sync, no tracking. Your data stays yours.

✅ **Free Forever**
No subscription, no hidden fees, no limits. Use it forever at no cost.

- ✅ ChatGPT (chatgpt.com)
- ✅ Claude (claude.ai)
- ✅ DeepSeek (chat.deepseek.com)
- ✅ Gemini (gemini.google.com)
- 🔄 Coming soon: Kimi, Microsoft Copilot, Perplexity

## How It Works

1. Install the extension
